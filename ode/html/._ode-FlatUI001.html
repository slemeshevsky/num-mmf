Задачи с начальными условиями для систем обыкновенных дифференциальных уравнений

Рассмотрим задачу Коши для системы обыкновенных дифференциальных уравнений $$ \begin{align} \tag{1} \frac{d u_i}{d t} &= f_i (t, u_1, u_2, \ldots, u_n), \quad t > 0\\ \tag{2} u_i(0) &= u_i^0, \quad i = 1, 2, \ldots, m. \end{align} $$

Используя векторные обозначения, задачу (1), (2) можно записать как задачу Коши $$ \begin{align} \tag{3} \frac{d \vec{u}}{d t} &= \vec{F}(t, \vec{u}), \quad t > 0, \\ \tag{4} \vec{u}(0) &= \vec{u}_0 \end{align} $$ В задаче Коши необходимо по известному решению в точке \( t = 0 \) необходимо найти из уравнения (3) решение при других \( t \).

Численные методы решения задачи Коши

Рассмотрим классические методы Рунге—Кутта и многошаговые методы решения задачи Коши для систем обыкновенных дифференциальных уравнений.

Методы Рунге—Кутта

При построении численных алгоритмов будем считать, что решение этой дифференциальной задачи существует, оно единственно и обладает необходимыми свойствами гладкости.

При численном решении задачи (1), (2) будем использовать равномерную, для простоты, сетку по переменной \( t \) с шагом \( \tau > 0 \): $$ \omega_\tau = \{ t_n = n\tau, \ n = 0, 1, \ldots\}. $$

Приближенное решение задачи (1), (2) в точке \( t = t_n \) обозначим \( \vec{y}^n \). Метод сходится в точке \( t_n \) , если \( |\vec{y}^n - \vec{u}(t_n)| \to 0 \) при \( \tau \to 0 \). Метод имеет \( p \)-ый порядок точности, если \( |\vec{y}^n - \vec{u}(t_n)| = O(\tau^p) \), \( p > 0 \) при \( \tau\to 0 \).

Одношаговый метод Рунге—Кутта в общем виде записывается следующим образом: $$ \begin{equation} \tag{5} \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} = \sum_{i = 1}^s b_i \vec{k}_i, \end{equation} $$ где $$ \begin{equation} \tag{6} \vec{k}_i = \vec{F}\left( t_n + c_i\tau, \vec{y}^n + \tau \sum_{j=1}^s a_{ij}\vec{k}_j \right), \quad i = 1, 2, \ldots, s. \end{equation} $$ Формула (5) основана на \( s \) вычислениях функции \( \vec{F} \) и называется \( s \)-стадийной. Если \( a_{ij} = 0 \) при \( j \geq i \) имеем явный метод Рунге—Кутта. Если \( a_{ij} = 0 \) при \( j > i \) и \( a_{ii} \ne 0 \), то \( \vec{k}_i \) определяется неявно из уравнения $$ \begin{equation} \tag{7} \vec{k}_i = \vec{F}\left( t_n + c_i\tau, \vec{y}^n + \tau \sum_{j=1}^{i-1} a_{ij}\vec{k}_j + \tau a_{ii} \vec{k}_i \right), \quad i = 1, 2, \ldots, s. \end{equation} $$ О таком методе Рунге—Кутта говорят как о диагонально-неявном.

Одним из наиболее распространенных является явный метод Рунге-Кутта четвертого порядка: $$ \begin{align*} \tag{8} \vec{k}_1 & = \vec{F}(t_n, \vec{y}^n), &\quad \vec{k}_2 &= \vec{F}\left( t_n + \frac{\tau}{2}, \vec{y}^n + \tau \frac{\vec{k}_1}{2} \right),\\ \vec{k}_3 &= \vec{F}\left( t_n + \frac{\tau}{2}, \vec{y}^n + \tau \frac{\vec{k}_2}{2} \right), &\quad \vec{k}_4 &= \vec{F}\left( t_n + \tau, \vec{y}^n + \tau \vec{k}_3 \right),\\ \frac{\vec{y}^{n+1} -\vec{y}^n}{\tau} &= \frac{1}{6} (\vec{k}_1 + 2\vec{k}_2 + 2\vec{k}_3 + \vec{k}_4) & & \end{align*} $$

Многошаговые методы

В методах Рунге—Кутта в вычислениях участвуют значения приближенного решения только в двух соседних узлах \( \vec{y}^n \) и \( \vec{y}^{n+1} \) — один шаг по переменной \( t \). Линейный \( m \)-шаговый разностный метод записывается в виде $$ \begin{equation} \tag{9} \frac{1}{\tau} \sum_{i=0}^m a_i \vec{y}^{n+1-i} = \sum_{i=0}^{m} b_i \vec{F}(t_{n+1-i}, \vec{y}^{n+1-i}), \quad n = m-1, m, \ldots \end{equation} $$ Вариант численного метода определяется заданием коэффициентов \( a_i \), \( b_i \), \( i = 0, 1, \ldots, m \), причем \( a_0 \ne 0 \). Для начала расчетов по рекуррентной формуле (9) необходимо задать \( m \) начальных значений \( \vec{y}^0, \vec{y}^1 , \ldots, \vec{y}^{m-1} \).

Различные варианты многошаговых методов (методы Адамса) решения задачи с начальными условиями для систем обыкновенных дифференциальных уравнений могут быть получены на основе использования квадратурных формул для правой части равенства $$ \begin{equation} \tag{10} \vec{u}(t_{n+1}) - \vec{u}(t_n) = \int_{t_n}^{t_{n+1}} \vec{F}(t, \vec{u}) dt \end{equation} $$

Для получения неявного многошагового метода используем для подынтегральной функции интерполяционную формулу по значениям функции \( \vec{F}^{n+1} = \vec{F}(t_{n+1}, \vec{y}^{n+1}) \), \( \vec{F}^n \), \dots, \( \vec{F}^{n+1-m} \), т.е. $$ \begin{equation} \tag{11} \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} = \sum_{i=0}^{m} b_i \vec{F}(t_{n+1-i}, \vec{y}^{n+1-i}) \end{equation} $$

Для интерполяционного метода Адамса (11) наивысший порядок аппроксимации равен \( m+1 \).

Для построения явных многошаговых методов можно использовать процедуру экстраполяции подынтегральной функции в правой части (10). В этом случае приближение осуществляется по значениям \( \vec{F}^n \), \( \vec{F}^{n-1} \), \dots, \( \vec{F}^{n+1-m} \) и поэтому $$ \begin{equation} \tag{12} \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} = \sum_{i=1}^{m} b_i \vec{F}(t_{n+1-i}, \vec{y}^{n+1-i}) \end{equation} $$

Для экстраполяционного метода Адамса (12) погрешность аппроксимации имеет \( m \)-ый порядок.

Примерами методов Адамса (11), (12) при \( m=3 \) являются $$ \begin{align} \tag{13} \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} &= \frac{1}{24} (9\vec{F}^{n+1} + 19\vec{F}^{n} - 5\vec{F}^{n-1} + \vec{F}^{n-2}),\\ \tag{14} \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} &= \frac{1}{12} (23 \vec{F}^{n} -16\vec{F}^{n-1} + 5\vec{F}^{n-2}), \end{align} $$ соответственно.

На основе методов Адамса строятся и схемы предиктор–корректор. На этапе предиктор используется явный метод Адамса, на этапе корректора — аналог неявного метода Адамса. Например, при использовании методов третьего порядка аппроксимации в соответствии с (14) для предсказания решения положим $$ \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} = \frac{1}{12} (23 \vec{F}^{n} -16\vec{F}^{n-1} + 5\vec{F}^{n-2}). $$ Для уточнеия решения (см. (13)) используется схема $$ \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} = \frac{1}{24} (9\vec{F}^{n+1} + 19\vec{F}^{n} - 5\vec{F}^{n-1} + \vec{F}^{n-2}). $$ Аналогично строятся и другие классы многошаговых методов.

Задачи

Задача 1: Метод Рунге—Кутта

Написать программу для численного решения задачи Коши для системы обыкновенных дифференциальных уравнений явным методом Рунге—Кутта четвертого порядка. Продемонстрировать работоспособность этой программы при решении задачи Коши (построить график зависимости решения от \( t \)) $$ \frac{d^2 u}{d t^2} = -\sin u, \quad 0 < t < 4\pi, \] \[ u(0) = 1, \quad \frac{d u}{d t} = 0. $$

Задача 2: Метод предиктор–корректор

Написать программу, которая реализует метод предиктор–корректор: $$ \begin{align*} \tag{15} \frac{\bar{\vec{y}}^{n+1} - \vec{y}^n}{\tau} &= \frac{1}{24} (55 \vec{F}^{n} -59\vec{F}^{n-1} + 37\vec{F}^{n-2} - 9 \vec{F}^{n-3}), \\ \frac{\vec{y}^{n+1} - \vec{y}^n}{\tau} &= \frac{1}{24} (9\vec{F}^{n+1} + 19\vec{F}^{n} - 5\vec{F}^{n-1} + \vec{F}^{n-2}). \end{align*} $$ С помощью этой программы решить задачу $$ \begin{align*} \frac{d y_1}{dt} &= y_2 - y_3, \\ \frac{d y_2}{dt} &= y_1 + a y_2,\\ \frac{d y_3}{dt} &= b + y_3(y_1 - c), \quad 0 < t \leq 100,\\ y_1(0) &= 1, \quad y_2(0) = 1, \quad y_3(0) = 1. \end{align*} $$ при \( a, b, c = 0.2, 0.2, 2.5 \) и \( a, b, c = 0.2, 0.2, 5 \).
